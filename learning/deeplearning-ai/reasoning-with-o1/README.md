# Reasoning with o1

> DeepLearning.AI Short Course | [Course Link](https://www.deeplearning.ai/short-courses/reasoning-with-o1/)

## Course Overview

Learn how to effectively prompt and use OpenAI's o1 model, which excels at abstract reasoning tasks through "test-time compute" and autonomous chain-of-thought reasoning.

## Key Concepts

- **Test-time compute**: o1 uses more computation at inference time to improve reasoning
- **Chain-of-thought**: Breaking problems into intermediate steps before answering
- **Orchestration**: Using o1 for planning, delegating execution to faster models (4o-mini)
